Inorder to run the PlacementTrackingSystem Project, please ensure you have the following:
* IDE: PyCharm (Community or Professional Edition)
* Python Version: Python 3.9 or above
* Internet Connection will be required only for cloning the GitHub repository



Steps to uncompress the PlacementTrackingSystem Project and make it work in Percy Gee Lab


Extracting Folder:
1. Copy the submitted compressed ZIP file to the local machine.
2. Right-click the ZIP file.
3. Select Extract All.
4. Choose a destination folder (e.g., Desktop or Documents).
5. Navigate to the extracted project folder.
6. Select the root project directory.
7. Click OK. 
8. Click Extract.

Opening in Pycharm:
1. Open PyCharm.
2. On the welcome screen, click Open.
3. Navigate to the extracted project folder.
4. Select the root project directory.
5. Click OK.

Python Compiler Details:
1. Python Interpreter Configuration
2. Go to File → Settings → Python Interpreter.
3. Select an existing interpreter OR
4. Create a new virtual environment:
5. Click Add Interpreter
6. Select Virtualenv Environment
7. Choose Python 3.9 or higher
8. Apply and confirm.

Inorder to run the development:
1. Navigate to the src directory.
2. Open main.py.
3. Right-click on main.py.
4. Select Run 'main'.
